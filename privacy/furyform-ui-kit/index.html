Privacy

FuryForm UI Kit privacy policy

The FuryForm UI Kit is a design system asset package. It does not collect personal data by itself, but it is designed to support privacy-conscious form experiences.

What this covers

  • The UI Kit itself is static and client-side only.
  • Data collection is controlled by the host application using the kit.
  • Any form behavior should make privacy transparent to users.

When to use it

Use the UI Kit for product experiences where forms are shown clearly, labels are explicit, and user trust is prioritized.
